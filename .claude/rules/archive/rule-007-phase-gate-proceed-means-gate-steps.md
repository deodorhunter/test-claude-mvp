# Rule 007 — "Proceed" at Phase Boundary Means Complete Gate Steps First

## Constraint

When the user says "proceed", "approved", or "continue" at a phase boundary, complete ALL 8 Phase Gate steps BEFORE reading, planning, or delegating anything for the next phase. "Proceed" is approval to run the Gate, not to start the next phase.

## Context

In Phase 2c, the Tech Lead immediately read next-phase US files upon approval — skipping the Phase Gate entirely (BACKLOG not updated, retrospective not run). The retrospective and DocWriter Mode B are mandatory even when zero incidents occurred. The complete 8-step gate checklist is in `.claude/agents/orchestrator.md` Phase 4 workflow.
