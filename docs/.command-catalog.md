# Slash Command Catalog

Quick reference for all 12 orchestration commands. Use this file to avoid re-generating command metadata during agent delegations (~3k tokens saved per delegation).

| Command | Purpose |
| `/consistency-check` | Score agent output for hallucinations (0-5 scale); blocks at ≤2. Run after each sub-agent returns. |
| `/deep-interview` | Socratic discovery: extract testable requirements from fuzzy intent via targeted questions. |
| `/handoff` | Parse git diff and append metrics + summary to docs/ARCHITECTURE_STATE.md. Run after merging US. |
| `/init-ai-reference` | Scan repo and generate docs/AI_REFERENCE.md (stack reference). Run once per major stack change. |
| `/judge` | Post-implementation verification: check git diff against US acceptance criteria (pass/fail per criterion). |
| `/learn` | Distill session insights into reusable notes; routes to expertise or workflow notes. Quality gate: non-googleable only. |
| `/notepad` | In-session institutional memory (Learnings, Decisions, Issues, Problems). Appends to docs/.session-notes.md. |
| `/phase-retrospective` | Mandatory end-of-phase report: incidents, rules, cost analysis. Append row to docs/SESSION_COSTS.md. |
| `/refine-backlog` | Pre-sprint refinement: apply yes-man filter to Backlog US in target phase. Produce verdict table. |
| `/reflexion` | Phase-gate ritual: extract 1-3 durable rules from session history into .claude/rules/project/. |
| `/review-session` | End-of-session checklist: audit docs for audience, EU AI Act coverage, citations, links. |

**Token savings:** ~3,000 tokens per agent delegation by referencing this catalog instead of re-generating command list at task time.
