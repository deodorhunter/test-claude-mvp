## Riepilogo

Mapping Plone roles → platform permissions (Manager/SiteAdmin → full access, Editor/Reviewer → query+mcp, Member → query only, unknown → DENY ALL). FastAPI dependency `require_permission(Permission.X)`. Enforcement solo da JWT — zero query DB per request.

**Residual risks:**
- Denial events loggati solo su stderr (non integrati con AuditService)
- Role refresh non riflessa fino a scadenza JWT (60min)
